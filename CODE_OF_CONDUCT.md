# Code of Conduct

## Our Standard

We want the SeeCost Tracker project to be a respectful, useful, and technically serious collaboration space.

Examples of behavior that contribute to a positive environment include:

- being respectful and direct
- giving actionable technical feedback
- assuming good intent
- focusing discussion on the code, behavior, and tradeoffs
- accepting correction gracefully

Examples of unacceptable behavior include:

- harassment, personal attacks, or discriminatory language
- bad-faith arguments or repeated antagonistic behavior
- publishing private information without permission
- derailing discussions away from the project work

## Enforcement

Project maintainers are responsible for clarifying and enforcing this code of conduct and may remove, edit, or reject comments, commits, issues, and pull requests that do not align with it.

## Reporting

If you experience or witness unacceptable behavior, contact the maintainers through a private channel or repository moderation flow.

## Scope

This code of conduct applies within project spaces, including issues, pull requests, discussions, and review threads.
